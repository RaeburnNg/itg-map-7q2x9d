§1 · The Vision 节目愿景

北极星 ——「让对的老板，遇见对的局」

表面是节目，本质是一个 双边局 + 线下变现平台。节目负责被看见 + 建信用，线下负责真正成局 + 变现 — 马来西亚第一个企业自媒体平台。
⚖ 双边局 · 谁入局
中小企业老板 — 主力观众 + 参赛者，要「入对的局」
大佬 / 投资人 — 高端观众 + 资源方，要「对的 dealflow」
一般商业观众 = 顺带覆盖，不主攻
🤝 线下变现 · 真正的产品
Roundtable — 圆桌高端对谈
私董会 — 入局老板 + 精选老板付费圈
招商局 — 大佬/投资人 ↔ 精选企业撮合现场
S1 · 立信
Proof
证格式 work + 人脉真兑现；记录首批成功案例 · 试跑第一场 roundtable
S2 · 起飞轮
Scale
信用变常设引擎：扩大佬池 · 开私董会 · 首场招商局 · 变现咬合
S3 · 制度化
Platform
入局成制度品牌：常设私董会/招商局 + 企业名册 · 可跨城 · 平台成真收入

§1 · The Gap 市场空间

大马的自媒体，全在 吃喝玩乐。
企业内容，是一片空地。

  • 福利、第一期、c.com 那类 — 都是吃喝玩乐 + lifestyle自媒体，充其量做新闻。
  • 没有人认真做企业内容 — 这就是《入局》看到的空间。
  • 企业节目要 professional 但不 boring：related to business，却不会让人想走。
  • 先发 + 把它做到这个深度 = 护城河。

§2 · The Mechanic 核心机制

报名 → 3 大佬 → 3 票 → 入局

RM 500k–1M
参赛资格 — 公司年营收门槛
3 票制
3 位大佬各一票，2 票以上 = 入局
人脉 ≠ 钱
奖励 = pre-arranged 高端商业人脉
报名时录下「你想连接哪一位大佬 / 哪类人脉」→ 大佬听 pitch、提问 interview → 投票 → 入局者获得一个饭局或会议，直接对接他想要的大佬 / 上市公司董事。

§2 · The Differentiator 护城河
国外节目，大佬投钱投资公司。
在大马，钱很难给、给也不多 — model 不 work。

《入局》用 人脉 当奖励货币。
这是为大马量身定的机制 — 也是没人抄得走的差异点。人脉资源真实存在（Norman 等大佬 + Jeff 的营收能力），奖励兑现得了：「一定会给到他才请他来。」
