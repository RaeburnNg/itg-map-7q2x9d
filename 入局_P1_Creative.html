① Show Outline — 一集 13min flow / beat sheet 锁 · rebuild w/ 团队

一集 ≈ 13 分钟(plan 中 10–12min),节奏「前快 · 中狠 · 后重」。下面是已锁的 beat sheet — 跟团队 在会上一格格 rebuild、对齐 run-of-show。每个 beat 后期会切成竖版金句弹药喂招募。

🎬 Beat Sheet per-episode · 13min

0:00–0:30
炸裂开场
Hook 点痛点 → 企业家入场。先勾观众「那可能是我」。
0:30–2:30
终极陈述
2min 路演 + montage — 企业家自我介绍 + 行业 + 想突破什么(节目要的是「想突破」的老板,不是「有问题」的老板)。
2:30–8:00
大老交锋 ★
5.5min · 核心高潮 — 3 大佬 Q&A 来回盘问。Norman 撕营销 · Shane 盘系统 · Tony 盘实业·开商脉。看企业家怎么接招。
8:00–10:00
博弈 + 宣判
3 大佬讨论是否出「金卡」→ 举牌 → ≥2 票 = 成功入局(金色「成功入局」VFX)。1 票或 0 票 = 出局。
10:00–13:00
人脉解密 ★
Connection Card「约见上市集团董事 / 大佬饭局」实体签名道具 → 企业家讲他想兑现的人脉 → 金句收尾。

🔧 机制护栏 guardrails

报名时前置人脉诉求(录下「你赢了想连接谁」)+ Connection Card 实体道具 = 奖励兑现的信用防线。drama 可 plan / 重排制造高潮(可 NG)。每集结尾还有一个调整中的 conversion section(轮值大佬给建议 / 钢轨片段)。
