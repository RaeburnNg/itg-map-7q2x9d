## ✏️ 怎么改(你是 collaborator)
1. 在 GitHub repo 直接编辑 `.html`(改文字 / typo / format)→ commit → GitHub Pages 约 1 分钟自动更新。
2. **改大的 / 改结构的**(加页、改导航、改框架)→ 先跟 Rae 讲一声,避免撞车。
3. 这些页是**自包含 HTML**(无外部依赖),改了直接生效;三种颜色模式(◐○◑)右上角切换。
